T summarize: An even function can be expanded in terms of cosine functions only, and an odd function can be expanded in terms of sine functions only.

When a function f⁢(t)𝑓𝑡 is given over an interval [0,L]0𝐿, several types of Fourier series expansion can be found, depending on the assumptions we make about the behavior function outside the interval. First, we can view [0,L]0𝐿 as the whole range for which f⁢(t)𝑓𝑡 is explicitly defined, and the function is extended periodically outside the interval. In this case, a conventional Fourier series results, which involves both the sine and cosine functions. Second, we can extend the function as an odd function onto the interval [−L,0]𝐿0, and compute the Fourier series for the extended function over [−L,L]𝐿𝐿. In this case, as we have just investigated in the above, a series of sine functions results. Finally, we can extend the function as an even function onto the interval [−L,0]𝐿0, and then compute the Fourier series of the extended function over the interval [−L,L]𝐿𝐿. In this case, as we have already seen, a series of cosine functions results. All these possibilities are explored in the example below.

Fourier expansions with sine or cosine functions only are called half-range expansions, because they only need the information about the function over half of the interval.
