Chapter 4 The Sturm-Liouville problems

4.1 Eigenvalues and eigenfunctions

In solving complex models, one often encounter problems that take the following form:

dd⁢x⁢[p⁢(x)⁢d⁢yd⁢x]+[q⁢(x)+λ⁢r⁢(x)]⁢y=0,a≤x≤b, (1)
α⁢y⁢(a)+β⁢y′⁢(a)=0, (2)
γ⁢y⁢(b)+δ⁢y′⁢(b)=0. (3)

In the above,

p⁢(x),q⁢(x),r⁢(x):given real functions,α,β,γ,δ:given real parameters, which could be zero,y⁢(x):an unknown function to be found,λ:an unknown parameter.

If p⁢(x)>0 and r⁢(x)>0 for ∀x∈[a,b], then problem (1)–(3) is called a regular Sturm–Liouville problem. Otherwise, it is a singular Sturm–Liouville problem.

Equations (2) & (3) dictates the behavior of y at the two end points, and are called boundary conditions. They are of the mixed type. Other possible boundary conditions are

{y⁢(a)=0,y⁢(b)=0,Dirichlet

or

{y′⁢(a)=0,y′⁢(b)=0,Neumann

or

{y⁢(a)=0,y′⁢(b)=0,or{y′⁢(a)=0,y⁢(b)=0.

These boundary conditions are special cases of (2)–(3).

Problem (1)–(3) are often encountered when applying separation of variables on partial differential equations.

We note that Eq. (1) is linear with a homogeneous forcing term. Thus y=0 is a trivial solution to Eq. (1). But can the problem admit non-trivial solutions? What values can λ take? The answer to the first question is yes, under certain conditions. Non-trivial solutions exists only if λ takes certain specific values. These specific values for λ are called eigenvalues (or characteristic values), and the corresponding non-trivial solutions are called eigen-functions (or characteristic functions).

We have the following results concerning eigenvalues and eigenfunctions.

Theorem 2.

The regular Sturm–Liouville problem (1)–(3) has infinitely many real and simple eigenvalues λn, n=0,1,2,…, which can be arranged in a monotonic sequence,

λ0<λ1<⋯

such that

limn→∞λn=∞.

Every eigenfunction yn⁢(x) associated with the corresponding eigenvalue λn has exactly n zeros in the interval (a,b). For each eigenvalue there is only one eigenfunction (up to a multiplicative constant).

Example 1

Solve the Sturm–Liouville problem

y′′+λ⁢y=0,0<x<π, (4)

with the boundary conditions

y⁢(0)=0andy⁢(π)=0. (5)

Solution: First, we consider the case when λ=0. Eq. (4) becomes

y′′=0,

whose solution must be of the form

y=A⁢x+B,A,B⁢ constants.

To determine the constant coefficients A and B, we apply the boundary conditions in (5), and find that A=B=0. Thus, in this case (λ=0), the only solution to (4)–(5) is

y=0,

which is the trivial solution, of no interest to us.

Next, we consider the case when λ<0. We write λ=−ξ2, for some ξ>0. In this case, Eq. (4) can be written as

y′′−ξ2⁢y=0. (6)

To solve this differential equation, we form its characteristic equation,

r2−ξ2=0,

which has roots

r=±ξ.

Thus, the differential eqn has as fundamental solutions

{eξ⁢x,e−ξ⁢x}.

Equivalently, the set of fundamental solutions can also be written as

{sinh⁡ξ⁢x,cosh⁡ξ⁢x}.

We recall that these hyperbolic functions are constructed from the exponential functions,

sinh⁡ξ⁢x=eξ⁢x−e−ξ⁢x2,cosh⁡ξ⁢x=eξ⁢x+e−ξ⁢x2.

Using the hyperbolic functions as building blocks, the general solution to (6) can be written as

y⁢(x)=A⁢cosh⁡ξ⁢x+B⁢sinh⁡ξ⁢x.

To determine the constant coefficients A & B, we again turn to the boundary conditions in (5),

y⁢(0) =A=0,
y⁢(π) =A⁢cosh⁡ξ⁢π+B⁢sinh⁡ξ⁢π=0⟹B=0.

Thus, in this case, y⁢(x)=0 is again the only solution.

Finally, we consider the case when λ>0. We write λ=ξ2 for some ξ>0. The equation (4) now has the form

y′′+ξ2⁢y=0. (7)

It has characteristic equation

r2+ξ2=0,

which has a pair of imaginary roots,

r=±i⁢ξ.

Thus, the differential equation (7) has fundamental solutions

{ei⁢ξ⁢x,e−i⁢ξ⁢x},

or equivalently,

{cos⁡ξ⁢x,sin⁡ξ⁢x}.

Of course, these two sets of functions are related via Euler’s identity,

cos⁡ξ⁢x=ei⁢ξ⁢x+e−i⁢ξ⁢x2,sin⁡ξ⁢x=ei⁢ξ⁢x−e−i⁢ξ⁢x2⁢i.

Using the sine-cosine functions as bases, the general solution to (7) can be written as

y⁢(x)=A⁢cos⁡ξ⁢x+B⁢sin⁡ξ⁢x.

To determine the constant coefficients A & B, we again invoke the boundary conditions in (5),

y⁢(0) =A=0,
y⁢(π) =A⁢cos⁡ξ⁢π+B⁢sin⁡ξ⁢π
=B⁢sin⁡ξ⁢π=0.

The coefficient B can take non-zero values only if

sin⁡ξ⁢π=0,

and this happens when ξ is an integer. We write

ξ=n,for some integer ⁢n≥1.

The eigenvalue λ is then given by

λ=ξ2=n2.

The non-zero solution (eigenfunction) takes the form

y=B⁢sin⁡n⁢x,

where B can be any value.

To summarize, the Sturm-Liouville problem (4)–(5) has non-trivial solutions (eigenfunctions) only when λ takes the value of positive integers, specifically, for each n≥1,

λn=n2,
yn=sin⁡n⁢x.

The arbitrary constant factor B has been dropped from the eigenfunction yn.

 

We did not prove Theorem 2 directly. Indirectly, we check its claims through the solutions to Example 1, and as it turns out, each one of the claims checks out:

  1. ✓

    There are infinitely many eigenvalues / eigenfunctions.

  2. ✓

    All the eigenvalues are real, positive, and the whole sequence is unbounded.

  3. ✓

    To each eigenvalue corresponds exactly one eigenfunction (up to a multiplicative constant).

  4. ✓

    With n≥1, the nth eigenfunction yn=sin⁡n⁢x has exactly n−1 zeros in the interval (0,π).

Example 2

Find the eigenvalues and eigenfunctions of the following problem

y′′+λ⁢y=0,0<x<π, (8)
y⁢(0)=0,y⁢(π)+y′⁢(π)=0. (9)

Solution: First, we consider the case when λ=0. Eq. (8) becomes

y′′=0,

whose solution must be of the form

y=A⁢x+B,A,B⁢ constants.

To determine the constant coefficients A and B, we apply the boundary conditions in (9), and find that

y⁢(0)=B=0,
y⁢(π)+y′⁢(π)=A⁢π+B+A=0⟹A=0.

Hence, when λ=0, the Sturm-Liouville problem (8)-(9) has only a trivial solution.

Next, we consider the case when λ<0. We write λ=−ξ2, for some ξ>0. In this case, Eq. (8) can be written as

y′′−ξ2⁢y=0. (11)

As in the previous example, the general solution to (11) can be written as

y⁢(x)=A⁢cosh⁡ξ⁢x+B⁢sinh⁡ξ⁢x.

To determine the constant coefficients A & B, we apply the boundary conditions in (9). By the first condition at x=0, we easily find that

y⁢(0)=A=0.

Thus,

y⁢(x)=B⁢sinh⁡ξ⁢x,y′⁢(x)=B⁢ξ⁢cosh⁡ξ⁢x.

By the second condition at x=π, we find that

y⁢(π)+y′⁢(π)=B⁢(sinh⁡ξ⁢π+ξ⁢cosh⁡ξ⁢π)=0.

We note that the expression inside the parentheses is strictly positive, because each individual term inside is strictly positive. For the equation to hold, B has no other choice but to be zero. Thus, in this case, y⁢(x)=0 is again the only solution.

Finally, we consider the case when λ>0. We write λ=ξ2 for some ξ>0. The equation (4) now has the form

y′′+ξ2⁢y=0. (12)

As in the previous example, the general solution to (12) is a linear combination of the sine-cosine basis functions,

y⁢(x)=A⁢cos⁡ξ⁢x+B⁢sin⁡ξ⁢x.

The first condition at x=0 in (9) mandates that

y⁢(0)=A=0.

Thus,

y⁢(x)=B⁢sin⁡ξ⁢x,and ⁢y′⁢(x)=B⁢ξ⁢cos⁡ξ⁢x.

By the second condition at x=π, we find that

y⁢(π)+y′⁢(π)=B⁢(sin⁡ξ⁢π+ξ⁢cos⁡ξ⁢π)=0.

The coefficient B can take non-zero values only if

sin⁡ξ⁢π+ξ⁢cos⁡ξ⁢π=0.

Does this equation of ξ have any solution? To find out, we rewrite this equation as follows,

ξ=−tan⁡ξ⁢π.

It is now clear that solutions to this equation are the intersections between functions ξ and −tan⁡ξ⁢π. We graph these two functions in Figure 4.1, and see that the curves for these two functions have infinitely many intersections over the positive range of the ξ-axis.

We let ξ1, ξ2, ⋯⁢⋯ be the sequence of positive real numbers that satisfy equation (13). For these ξ values,

λ=ξk2,k≥1,

and the Sturm-Liouville problem (8)-(9) adopts non-trivial solutions in the form of

y⁢(x)=B⁢sin⁡ξk⁢x,B⁢ arbitrary.

To summarize, the Sturm-Liouville problem (8)–(9) has eigenvlaues and eigenfunctions in the form of

λk=ξk2,ξk⁢ the kth solution of (13),
yk=sin⁡ξk⁢x.

As usual, the arbitrary constant factor has been dropped from the eigenfunction.
